# Data

This document describes how to download each dataset and how the files are organized under `./data`. Here is an example of a data structure.

```text
./data
├── EgoDex
│   ├── download/   # Original compressed archives
│   ├── raw/        # Unpacked raw data
│   └── processed/  # Data generated by our preprocessing scripts
└── ...
```

---

## EgoDex

### Download

Download EgoDex dataset from [official website](https://github.com/apple/ml-egodex) or using following commands.
```bash
mkdir -p data/EgoDex/{download,raw,processed}
cd data/EgoDex/download

files=(test extra part1 part2 part3 part4 part5)
base_url="https://ml-site.cdn-apple.com/datasets/egodex"

for name in "${files[@]}"; do
  zip_file="${name}.zip"
  url="${base_url}/${zip_file}"

  curl -L --fail --retry 3 -o "${zip_file}" "${url}"
  unzip -o "${zip_file}" -d ../raw
 done
```

### Structure

```text
./data/EgoDex/raw
├── part1/
├── part2/
├── part3/
├── part4/
├── part5/
├── extra/
└── test/
```
